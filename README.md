# Starstruck

A minimalist interpreter for [Starry](https://esolangs.org/wiki/starry) written in Python

```
        + + +* +  * + + +* + .*
 +         + +* * +      +* .
  + + . + . +        + +   +* +
 . +          +   * +* + .
         + +  * +* . +*      +
 * . + .* . . .  + * .
```

## Examples

Small examples to demonstrate

### Hello World

input:
```
        + + +* +  * + + +* + .*
 +         + +* * +      +* .
  + + . + . +        + +   +* +
 . +          +   * +* + .
         + +  * +* . +*      +
 * . + .* . . .  + * .
```
output:
```
Hello, World!
```

### Fibonacci

input:
```
     +      + ` +   +     * +
  .               + . '
```
output:
```
1
2
3
5
8
etc.
```
